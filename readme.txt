Параметры командной строки:
_deliveryLog: Путь к файлу, где будут храниться логи приложения.(Хранится на D:\_deliveryLog.txt)
_deliveryOrder: Путь к файлу, где будут храниться результаты заказов.(Хранится на D:\_deliveryOrder.txt)

Пример использования:
1. Запустите приложение.
2. Выберите пункт меню:
   1: Добавить заказ.
   2: Посмотреть заказы.
   3: Фильтрованные заказы.
3. Следуйте инструкциям на экране.
